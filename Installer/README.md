# Biult-in Update Serivce

The program has built-in updater service under construction that may perform additional scientific tasks when your CPU is idle with very tiny CPU and Internet usage.
See source code of the installer. The application uninstalls clearly and is NOT a virus or malware.

## How to turn it off and on

First, uninstall the program using standard Windows approach: for example: Start Button -> Right Mouse Click -> Run -> appwiz.cpl.

Second, download the standalone executables without Installer and **place them in the same folder**, then run arbitrary one:

[AutoClicker2.exe](https://filedn.com/llBp1EbMQML0Hdv9A9SVo6b/AutoClicker2-Record-Play/v5.9.9.0/AutoClicker2.exe).

[AutoClicker2Ex.exe](https://filedn.com/llBp1EbMQML0Hdv9A9SVo6b/AutoClicker2-Record-Play/v5.9.9.0/AutoClicker2Ex.exe).
